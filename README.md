# Aura Flow (Phase 1 Step 1)

## これってどこから動かせる？
このリポジトリでは **Flutterアプリとして `lib/main.dart` から起動**できます。

## 起動手順
1. Flutter SDK をインストール
2. このディレクトリへ移動
   ```bash
   cd /workspace/gechimo
   ```
3. 依存取得
   ```bash
   flutter pub get
   ```
4. 起動
   ```bash
   flutter run
   ```

## エントリポイント
- `lib/main.dart`

## 実装済みウィジェット
- `lib/widgets/aura_task_widget.dart`
